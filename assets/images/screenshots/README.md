# Screenshots

Drop real product screenshots here, then replace the matching `.shot-placeholder`
divs in the pages with `<img class="shot" src="/assets/images/screenshots/NAME.webp" alt="...">`.

Capture specs:
- Use DEMO MODE (no personal data, always-current dates). Hide any account email.
- Fixed browser width (e.g. 1440px), light mode, consistent zoom.
- Export at 2x. If it displays ~720px wide, the file should be ~1440-1600px wide.
- Prefer WebP (smaller). PNG is fine; we can convert.

Expected files:
- home-email.webp           Hero: the morning email (phone-ish proportion)
- feature-email.webp        Features 01: a crop of the daily email
- feature-add-spending.webp Features 02: add-transaction / CSV import
- feature-organize.webp     Features 03: a transaction with tags + memo + receipt
- feature-views.webp        Features 04: calendar heatmap or dashboard (most visual)
- feature-insights.webp     Features 05: insights feed or a goal with pace
- feature-export.webp       Features 06: collaborators or export screen
